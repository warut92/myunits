# ทัศนศิลป์ 3

รูปแบบทัศนธาตุและแนวคิดในงานทัศนศิลป์

การวาดภาพสื่อความหมายและเรื่องราว

การวาดภาพถ่ายทอดบุคลิกของตัวละคร

งานทัศนศิลป์ในการโฆษณา

# ดนตรีไทยพื้นฐาน
